The Text to Story Narration application is an AI-driven tool that transforms a text prompt into a narrated story with visuals. It uses Cohere's NLP for story generation, Selenium for automated image creation, gTTS for voice narration, and MoviePy for video compilation. The final output includes an animated GIF and a video, providing an immersive storytelling experience.
